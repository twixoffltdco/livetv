# 📺 LiveM3U - Собственный поисковой робот IPTV каналов РФ

**Независимый проект, который НЕ использует iptv-org или другие готовые списки!**

## 🚀 Что это?

LiveM3U - это автономный поисковой робот, который:
- **САМ ищет** рабочие IPTV потоки российских телеканалов
- **Проверяет** доступность каждого найденного потока
- **Создаёт** актуальный M3U плейлист
- **Обновляет** плейлист автоматически каждый час

## 🔥 Ключевые особенности

✅ **Полностью независимый** - не используем iptv-org, zabava-project или другие готовые каталоги  
✅ **Умный поиск** - сканирует провайдеров РФ по известным паттернам  
✅ **IP сканер** - может сканировать диапазоны IP адресов на наличие IPTV потоков  
✅ **Автообновление** - работает 24/7 в фоновом режиме  
✅ **Только рабочие** - умная фильтрация нерабочих ссылок  
✅ **РФ фокус** - приоритет российским телеканалам  
✅ **Тематические плейлисты** - автоматическое разделение по категориям после каждого обновления 🆕  
✅ **Авто-категоризация** - новые каналы автоматически добавляются в соответствующие тематические плейлисты

## 🛠️ Установка и запуск

### Требования
- Python 3.8+
- Библиотека `aiohttp` (`pip install aiohttp`)

### Быстрый старт

```bash
# Однократный запуск поиска
python src/bot.py --once

# Непрерывный режим (проверка каждый час)
python src/bot.py --interval 3600

# С сканированием IP диапазонов
python src/bot.py --ip-ranges 195.161.0.0/16 95.161.0.0/16
```

## 🌐 Как использовать плейлист

### Вариант 1: Полный плейлист (все каналы)
⚠️ **Внимание:** Полный плейлист содержит 7900+ каналов и может медленно работать в некоторых плеерах.

1. **Скачайте** файл `/data/playlist.m3u`
2. **Или используйте URL**: `https://your-domain.com/data/playlist.m3u`
3. **Откройте** в любом IPTV плеере (VLC, Kodi, IPTV Org, TiviMate и др.)

### Вариант 2: Веб-плеер онлайн 🆕
Смотрите IPTV прямо в браузере без установки дополнительных программ!

1. **Откройте** `https://your-domain.com/live/player.html`
2. **Выберите** полный плейлист или тематическую категорию
3. **Кликните** на любой канал для воспроизведения

**Возможности веб-плеера:**
- ✅ Поддержка HLS потоков (.m3u8)
- ✅ Загрузка полного плейлиста (7900+ каналов)
- ✅ Выбор тематических плейлистов из 170+ категорий
- ✅ Поиск каналов по названию
- ✅ Загрузка локальных M3U файлов
- ✅ Красивый современный интерфейс

### Как выбрать плейлист?

1. **Для быстрого старта:** Используйте `Основные.m3u` (102 канала) - только главные каналы РФ
2. **Для максимального охвата:** Используйте `IPTV.m3u` (3278 каналов) - баланс между размером и количеством
3. **Для конкретного региона:** Выберите свой город (например, `Москва.m3u`)
4. **Для зарубежных каналов:** Выберите страну или язык
5. **Просмотреть все категории:** Откройте `/data/playlists/INDEX.m3u` для просмотра всех 170+ категорий
6. **Для всех каналов сразу:** Используйте полный `playlist.m3u` (может тормозить)

💡 **Совет:** Тематические плейлисты обновляются автоматически вместе с основным плейлистом каждые 30 минут. Новые каналы сразу попадают в нужную категорию!

## 🔍 Как работает поисковой робот

### 1. Поиск по провайдерам
Робот проверяет популярные домены российских провайдеров:
- `rostelecom.ru`, `ertelecom.ru`, `domru.ru`
- `megafon.ru`, `beeline.ru`, `mts.ru`
- `tvzavr.ru`, `peers.tv`, `1tv.ru`, `smotrim.ru`
- И другие...

### 2. Проверка паттернов
Для каждого домена проверяются типичные пути:
- `/stream.m3u8`, `/live.m3u8`, `/iptv.m3u8`
- `/hls/stream.m3u8`, `/channel.m3u8`
- И варианты с `.mpd`, `.m3u`

### 3. Сканирование IP диапазонов (опционально)
Можно указать диапазоны IP для прямого сканирования:
```bash
python src/bot.py --ip-ranges 195.161.0.0/16 95.161.0.0/16
```

### 4. Умная валидация
Каждый найденный поток проверяется:
- ✅ Content-Type заголовки (mpegurl, dash+xml)
- ✅ Сигнатуры M3U8/MPD в содержимом
- ✅ Доступность URL
- ✅ Принадлежность к РФ (по ключевым словам)

### 5. 🆕 Автоматическое создание тематических плейлистов
После каждого сканирования робот автоматически:
- Разделяет все каналы по категориям (`group-title`)
- Создаёт отдельные M3U файлы для каждой категории
- Обновляет индексный файл `INDEX.m3u`
- Новые каналы сразу добавляются в соответствующие тематические плейлисты

Это решает проблему производительности при большом количестве каналов (20000+) и позволяет пользователям выбирать только нужные категории!

## 📊 Формат выходных данных

### playlist.m3u
```m3u
#EXTM3U
# Обновлён: 2024-01-15 14:30:00
# Всего каналов: 50
# Сгенерировано собственным поисковым роботом LiveM3U

#EXTINF:-1 tvg-name="Первый канал" group-title="RU",Первый канал
https://example.com/stream.m3u8

#EXTINF:-1 tvg-name="Россия 1" group-title="RU",Россия 1
https://example.com/live.m3u8
```

### found_streams.json
```json
{
  "https://example.com/stream.m3u8": {
    "name": "Первый канал",
    "url": "https://example.com/stream.m3u8",
    "found_at": "2024-01-15T14:30:00",
    "country": "RU"
  }
}
```
