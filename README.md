# 📺 LiveM3U - Собственный поисковой робот IPTV каналов РФ

**Независимый проект, который НЕ использует iptv-org или другие готовые списки!**

## 🚀 Что это?

LiveM3U - это автономный поисковой робот, который:
- **САМ ищет** рабочие IPTV потоки российских телеканалов
- **Проверяет** доступность каждого найденного потока
- **Создаёт** актуальный M3U плейлист
- **Обновляет** плейлист автоматически каждый час

## 🔥 Ключевые особенности

✅ **Полностью независимый** - не используем iptv-org, zabava-project или другие готовые каталоги  
✅ **Умный поиск** - сканирует провайдеров РФ по известным паттернам  
✅ **IP сканер** - может сканировать диапазоны IP адресов на наличие IPTV потоков  
✅ **Автообновление** - работает 24/7 в фоновом режиме  
✅ **Только рабочие** - умная фильтрация нерабочих ссылок  
✅ **РФ фокус** - приоритет российским телеканалам

## 🛠️ Установка и запуск

### Требования
- Python 3.8+
- Библиотека `aiohttp` (`pip install aiohttp`)

### Быстрый старт

```bash
# Однократный запуск поиска
python src/bot.py --once

# Непрерывный режим (проверка каждый час)
python src/bot.py --interval 3600

# С сканированием IP диапазонов
python src/bot.py --ip-ranges 195.161.0.0/16 95.161.0.0/16
```

## 🌐 Как использовать плейлист

1. **Скачайте** файл `/data/playlist.m3u`
2. **Или используйте URL**: `https://your-domain.com/data/playlist.m3u`
3. **Откройте** в любом IPTV плеере:
   - VLC Media Player
   - Kodi
   - Perfect Player
   - IPTV Smarters
   - Smart TV приложения

## 🔍 Как работает поисковой робот

### 1. Поиск по провайдерам
Робот проверяет популярные домены российских провайдеров:
- `rostelecom.ru`, `ertelecom.ru`, `domru.ru`
- `megafon.ru`, `beeline.ru`, `mts.ru`
- `tvzavr.ru`, `peers.tv`, `1tv.ru`, `smotrim.ru`
- И другие...

### 2. Проверка паттернов
Для каждого домена проверяются типичные пути:
- `/stream.m3u8`, `/live.m3u8`, `/iptv.m3u8`
- `/hls/stream.m3u8`, `/channel.m3u8`
- И варианты с `.mpd`, `.m3u`

### 3. Сканирование IP диапазонов (опционально)
Можно указать диапазоны IP для прямого сканирования:
```bash
python src/bot.py --ip-ranges 195.161.0.0/16 95.161.0.0/16
```

### 4. Умная валидация
Каждый найденный поток проверяется:
- ✅ Content-Type заголовки (mpegurl, dash+xml)
- ✅ Сигнатуры M3U8/MPD в содержимом
- ✅ Доступность URL
- ✅ Принадлежность к РФ (по ключевым словам)

## 📊 Формат выходных данных

### playlist.m3u
```m3u

# Обновлён: 2024-01-15 14:30:00
# Всего каналов: 50
# Сгенерировано собственным поисковым роботом LiveM3U

#EXTINF:-1 tvg-name="Первый канал" group-title="RU",Первый канал
https://example.com/stream.m3u8

#EXTINF:-1 tvg-name="Россия 1" group-title="RU",Россия 1
https://example.com/live.m3u8
```

```json
{
  "https://example.com/stream.m3u8": {

    "url": "https://example.com/stream.m3u8",
    "found_at": "2024-01-15T14:30:00",
    "country": "RU"
  }
}
```
