## 📁 Как использовать

### Вариант 1: Индексный файл
Загрузите `INDEX.m3u` - это оглавление всех плейлистов с ссылками на категории.

### Вариант 2: Конкретная категория
Выберите нужный файл из папки `/data/playlists/`:
- Для быстрого старта: `Основные.m3u` (102 канала)
- Для максимального охвата: `IPTV.m3u` (3278 каналов)
- Для региона: выберите свой город
- Для зарубежья: выберите страну

### Вариант 3: Несколько категорий
Загрузите несколько файлов в плеер одновременно:
```
Основные.m3u + Москва.m3u + News.m3u
```

## 💡 Рекомендации

### ✅ Используйте тематические плейлисты если:
- У вас старый Smart TV или приставка
- Плеер тормозит с большим количеством каналов
- Вам нужны только каналы определённой тематики
- Хотите быстрее переключаться между каналами

### ⚠️ Полный плейлист может понадобиться если:
- Вы хотите иметь все каналы в одном списке
- Ваш плеер хорошо работает с большими файлами
- У вас мощное устройство (PC, современный Smart TV)

## 🔗 Ссылки

- **Полный плейлист:** `/data/playlist.m3u`
- **Индекс категорий:** `/data/playlists/INDEX.m3u`
- **Все категории:** `/data/playlists/`

## 🔄 Обновление

Тематические плейлисты обновляются автоматически при каждом запуске робота:
```bash
python src/bot.py --interval 1800
```

После обновления запустите скрипт разделения:
```bash
python split_playlist.py
```
